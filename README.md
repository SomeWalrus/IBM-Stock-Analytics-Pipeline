# Stock Market Data Pipeline & Analysis

Automated ETL pipeline that collects, stores, and analyzes daily stock market data using Python, Alpha Vantage API, SQLite, and GitHub Actions.

---

## Overview

This project implements an automated data pipeline that retrieves daily stock market data using the Alpha Vantage API, processes and stores it in a SQLite database, and performs exploratory time-series analysis and visualization using Python.

The pipeline is scheduled to run daily using GitHub Actions, ensuring continuously updated data for analysis.

---

## Features

- Automated daily data collection using GitHub Actions
- API-based extraction of stock market data (Alpha Vantage)
- Data cleaning and transformation using pandas
- Storage and querying using SQLite
- Time-series visualization of stock price trends in Jupyter Notebook

---

## Tech Stack

- Python
- pandas
- SQLite
- matplotlib
- Jupyter Notebook
- GitHub Actions
- Alpha Vantage API
